ACM协会简介

ACM工作室隶属电子信息工程学院，是学校为了应对一年一度的acm程序设计大赛而成立的社团，ACM竞赛规定每支参赛队伍由三名在校大学生组成，他们需要在规定的五个小时内解决八个或更多的复杂实际编程问题。每队使用一台电脑，参赛者争分夺秒，与其他参赛队伍拼比逻辑、策略和心理素质。


介于加入qq群的各位同学中，有非电院、文理学院，以及之前并未接触过的同学，可能还不是很了解编程，我们知道，生活中我们所运用的到程序是由数据和指令按照一定的顺序组合而成的，而把程序和指令按照特定排列组合的这一过程，我们就称为编程


再用一种简单的方法来让同学们理解，同学们应该都用过电脑吧，电脑中有一个黑框框之类的窗口，我们称之为命令提示符，如果我们想在这个黑窗口中输入一个心形，就像这样，我们应该怎么做呢!


其实很简单,

#include <stdio.h>

int main() {
    for (float y = 1.5f; y > -1.5f; y -= 0.1f) {
        for (float x = -1.5f; x < 1.5f; x += 0.05f) {
            float a = x * x + y * y - 1;
            putchar(a * a * a - x * x * y * y * y <= 0.0f ? '*' : ' ');
        }
        putchar('\n');
    }
}


是不是很简单，仅仅需要几行代码，就能做出一个这么漂亮的心形

但是如果我们要增加点难度呢，比如让心形动起来，又要怎么做呢？



这就牵涉到我们编程的核心了，那就是各种——算法，算法（Algorithm）是指解题方案的准确而完整的描述，是一系列解决问题的清晰指令，算法代表着用系统的方法描述解决问题的策略机制。运用算法，我们能让计算机帮我们解决各种需要解决的问题


熟练运用各种算法解决问题，这就是ACM竞赛的目的核心所在，不仅如此，如果我们将来从事软件开发工作，算法应用的好坏也是衡量一个程序性能的关键所在，可以说，算法是学习各种语言的基础


在工作室中，学长学姐们可以为各位提供学习所需要的必须资料，以及为同学们的疑问答疑，在工作室的学习也许是枯燥的，但是同学们不要忘了我们来上这个大学并不是来享受的，往届很多学长学姐的经历告诉我们，那些从大一开始到结束并未加入任何学习性质社团的同学额，他们的学习总是很曲折的，上课听不懂老师讲的内容，就感觉烦躁，然后整天打游戏度日，结果大学四年就这么荒废过去，而反观在工作室学习的同学们，由于他们很早就有接触这些专业知识，上课的时候感觉一点也不吃力，并且很早就对自己的发展前景有了规范，不会上了一年学还不知到要做什么，所以说，加入一个学习性质的社团是很有必要的



——在工作室学什么？
——同学们主要学习内容是C/C++语言，后期会学习数据结构和算法相关的知识


——怎么学习
——工作室的学长学姐们会为同学们提供各种学习资料，并且会对学习过程中遇到的问题进行帮助解决


由于工作室是学习性质的社团，抛开社团这一性质来说，工作是类似于学习小组之类的团体，所以，工作室是不可以进行娱乐活动的，并且也会有时间要求，希望同学们能提前做好心里预期
